**放行WAF回源IP**

  您的网站成功接入WAF防护后，访问您网站的所有请求将先流转到WAF实例，经WAF实例过滤后再返回到源站服务器。流量经WAF实例返回源站的操作称为回源。
由于WAF实例的IP数量有限，源站服务器收到的所有请求都来自这些IP，在源站服务器上的安全软件（如安全狗、云锁）看来，这种情况是很可疑的，可能会屏蔽回源IP。因此，在接入WAF防护后，您需要在源站服务器的安全软件上设置放行所有WAF回源IP。
